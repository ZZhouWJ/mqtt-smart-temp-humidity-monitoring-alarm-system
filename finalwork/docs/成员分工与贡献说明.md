# 成员分工与贡献说明

## 成员 A：数据模拟与后端业务引擎

成员 A 主要负责系统的数据链路与后端处理逻辑，具体工作包括：

1. 编写 Python 温湿度传感器模拟程序；
2. 生成带随机噪声的温度、湿度数据；
3. 模拟跳变异常、丢包异常和短暂离线；
4. 设计 MQTT Topic 与 JSON 数据格式；
5. 实现 MQTT 数据发布与断线自动重连；
6. 搭建 Spring Boot 后端项目；
7. 实现 MQTT 数据订阅与 JSON 解析；
8. 实现滑动平均滤波；
9. 实现跳变异常检测和丢包检测；
10. 实现超温、超湿报警判断；
11. 通过 WebSocket 将处理后的数据推送给前端；
12. 编写后端运行说明和核心技术说明。

## 成员 B：前端可视化与项目汇报设计

成员 B 主要负责前端展示、交互效果和项目汇报材料，具体工作包括：

1. 搭建 Vue + Vite 前端项目；
2. 使用 ECharts 绘制温湿度实时曲线；
3. 通过 WebSocket 接收后端实时数据；
4. 设计当前温度、湿度、设备状态、报警状态卡片；
5. 实现超温、超湿报警的前端视觉提示；
6. 展示最近报警记录和最近数据列表；
7. 整理系统截图和演示流程；
8. 撰写项目 PPT 汇报稿；
9. 整理前端运行说明；
10. 负责最终压缩包结构整理。

## 共同完成内容

1. MQTT Topic 和 JSON 字段对齐；
2. 后端 WebSocket 推送格式与前端接收格式对齐；
3. 系统联调测试；
4. 异常数据和报警功能测试；
5. 断线重连测试；
6. 代码注释检查；
7. 答辩演示排练；
8. 最终项目打包提交。
